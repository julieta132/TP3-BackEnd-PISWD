# TP3-BackEnd-PISWD
Trabajo Práctico final de PISWD
Desarrollo Back-End del módulo CRUD de alumnos para la intranet institucional 
# Carpetas
# SQL
Contiene los archivos .sql para la creación del esquema de la base de datos
- CrearBDDYTablas.sql: Crea base de datos y tablas del sistema
- InsertarDatos.sql: Agregar registros en las tablas
- Procedimientos.sql: Realizar procedimientos almacenados para el CRUD
- Consultas.sql: Realizar consultas a la base de datos
# PHP
Contiene todos los archivos PHP del proyecto
- TodosUsuarios.php: Es el archivo con código html donde se muestra en una tabla todos los usuarios y las accciones a realizar
- AltaUsuario.php: Formulario html para agregar un usuario nuevo
- ModificarUsuario.php: Formulario html que muestra los datos del usuario seleccionado y permite modificarlos
- BajaUsuario.php: Muestra los datos del usuario seleccionado con un botón para eliminar
- FuncionesBDD.php: Incluye las funciones principales de conexión al servidor y a la base de datos
# IMG
Incluye las imagenes utilizadas 
- logo.fjij: El logo institucional
# CSS
Contiene los archivos css para los estilos de la página
- estilos.css: Los estilos principales de la página
